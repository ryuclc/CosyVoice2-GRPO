# CosyVoice2-GRPO
A simple implementation for improving CosyVoice2 by GRPO method

This is the code of paper “Group Relative Policy Optimization for Text-to-Speech with Large Language Models”

We modify official CosyVoice2 code to achieve GRPO fine-tune, which is inspired by huggingface trl, you do not need to install any other modules or learn any other RL frameworks. Get started quickly.
